# JSON File Visualizer

A powerful web-based tool designed to visualize JSON data structures in interactive and readable formats.

## Features

- **Interactive Visualization**: Visualize your data in a clear, interactive tree format.
- **Search Capabilities**: Integrated search functionality to quickly find keys or values within large JSON files.
- **Node Management**:
    - Expand and collapse nodes to manage complexity.
    - Smooth animations for better user experience.
- **Analytics Support**: Capable of handling complex analytics data structures.
- **Performance**: Optimized for handling various JSON file sizes.

## Getting Started

### Prerequisites

Ensure you have Node.js installed on your system.

### Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application

Start the development server:

```bash
npm run dev
```

The application will launch in your default browser (usually at http://localhost:5173 or similar).

## Usage

- Load your JSON files to visualize them.
- Use the search bar to filter properties.
- Click on nodes to potential expand/collapse detailed sections.
